# TrafficMagnit

## Setup
Clone project
    
    git clone https://github.com/vblacksquare/TrafficMagnit

Fill your .env file
    
    cp exmaple.env .env

Fill your docker-compose.yml

Start project
    
    docker compose up --build -d

## Usage
- Docs: http://127.0.0.1:8000/api/docs/swagger
- Admin: http://127.0.0.1:8000/admin
- Management: python manage.py export
